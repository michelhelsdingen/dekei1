<task type="auto">
  <name>Task 2: Cloudflare DNS configureren voor dekei1.ltcdekei.nl</name>
  <files></files>
  <action>
    Voeg een DNS A record toe in Cloudflare voor dekei1.ltcdekei.nl → 37.97.202.216 (plex VPS).

    Stap 1 — Zoek de Cloudflare zone ID voor ltcdekei.nl:
    ```bash
    curl -s -X GET "https://api.cloudflare.com/client/v4/zones?name=ltcdekei.nl" \
      -H "Authorization: Bearer ec80358daa6f073203b69cc1fdad2bcc2e7fc" \
      -H "Content-Type: application/json" | jq '.result[0].id'
    ```
    Als ltcdekei.nl niet gevonden wordt (empty result), probeer dan:
    ```bash
    curl -s -X GET "https://api.cloudflare.com/client/v4/zones" \
      -H "Authorization: Bearer ec80358daa6f073203b69cc1fdad2bcc2e7fc" \
      -H "Content-Type: application/json" | jq '.result[] | {name:.name, id:.id}'
    ```
    Dit toont alle zones — zoek de zone die ltcdekei.nl of ltcdekei omvat.

    Stap 2 — Sla ZONE_ID op en voeg A record toe:
    ```bash
    ZONE_ID="<zone_id_from_step_1>"
    curl -s -X POST "https://api.cloudflare.com/client/v4/zones/$ZONE_ID/dns_records" \
      -H "Authorization: Bearer ec80358daa6f073203b69cc1fdad2bcc2e7fc" \
      -H "Content-Type: application/json" \
      --data '{
        "type": "A",
        "name": "dekei1",
        "content": "37.97.202.216",
        "ttl": 1,
        "proxied": false
      }'
    ```
    Let op: `proxied: false` — Coolify handelt SSL af, geen Cloudflare proxy nodig.

    Stap 3 — Controleer of het record aangemaakt is:
    ```bash
    curl -s "https://api.cloudflare.com/client/v4/zones/$ZONE_ID/dns_records?name=dekei1.ltcdekei.nl" \
      -H "Authorization: Bearer ec80358daa6f073203b69cc1fdad2bcc2e7fc" | jq '.result[0] | {name, content, type}'
    ```

    Als het zone niet gevonden wordt: rapporteer de volledige lijst van zones zodat de user het juiste domein kan aanwijzen.
  </action>
  <verify>
    # DNS propagatie check (kan 1-5 minuten duren):
    dig dekei1.ltcdekei.nl A +short
    # Verwacht: 37.97.202.216
  </verify>
  <done>
    DNS A record 'dekei1.ltcdekei.nl → 37.97.202.216' staat in Cloudflare. dig of nslookup bevestigt het IP.
  </done>
</task>
